This project creates a dashboard out of my music listening history collected through Listenbrainz. csv_generate.py makes use of the Listebrainz API in order to import my listening history and generate a csv file with the pandas library for use in Power BI.  
&emsp;The csv files output from this script are:
  
    - listen_history.csv containing track by track listen history.
    - artists.csv containing all of the artists for each track.
    - genres.csv containing all of the generes for each track.
